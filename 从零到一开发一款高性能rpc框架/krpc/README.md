# krpc

## 目录

``` 
├── client   // 支持客户端发包
├── codec    // 自定义协议的解析、序列化和反序列化
├── interceptor  // 提供框架拦截器能力
├── logger // 提供日志能力
├── metadata // 提供客户端和服务端参数传递能力
├── plugin // 提供第三方插件化支持能力
├── pool // 池技术，支持连接池、对象池等实现，提供客户端连接的复用、对象复用等能力
├── protocol // 提供自定义私有协议能力
├── selector // 提供寻址能力，支持服务发现（resolver）、负载均衡 （loadbalance）等默认实现
├── server // 支持服务端收包
├── stream // 提供客户端和服务端上下文数据透传能力，后续还会支持流式传输
└── transport // 提供底层的通信能力
```
